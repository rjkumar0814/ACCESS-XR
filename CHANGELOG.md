# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project follows [Semantic Versioning](https://semver.org/).

---

## [1.0.0] - 2026-01-01

### Added

- Initial public release of ACCESS-XR.
- Unified multimodal communication framework for Extended Reality (XR).
- Lightweight attention-enhanced YOLOv8 gesture recognition module.
- Whisper-based automatic speech recognition with entropy-aware confidence estimation.
- Fuzzy semantic mapping for multimodal translation.
- Uncertainty-aware decision-level fusion strategy.
- FastAPI backend for multimodal inference.
- React Native frontend for mobile XR interaction.
- Modular architecture supporting edge and cloud deployment.
- Documentation for installation, usage, API, and reproducibility.
- Initial pretrained model weights.
- Example datasets and configuration files.

### Experimental Features

- Gesture-to-Text translation.
- Speech-to-Sign translation.
- Text-to-Sign translation.
- Accessibility-aware rendering pipeline.
- Latency-aware multimodal communication.

### Documentation

- Added comprehensive README.
- Added architecture documentation.
- Added reproducibility guidelines.
- Added API documentation.
- Added model documentation.
